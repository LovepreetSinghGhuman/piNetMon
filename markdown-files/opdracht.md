# Eindopdracht Opgave

## Minimum benodigdheden

* Minstens één sensor die data vastlegt.
* De data moet **lokaal** kunnen worden opgeslagen.
* De data moet kunnen worden doorgestuurd naar de cloud.
* De data moet **in de cloud** worden opgeslagen.
* Er moet minstens één **lokaal AI-model** aanwezig zijn.
* Er moet minstens één **AI-model in de cloud** draaien.
* De gebruiker moet kunnen **interageren** met het IoT-device.
* Het moet mogelijk zijn om de **configuratie van het toestel vanop afstand** te bekijken en aan te passen.
* Er moet een **dashboard** (streamlit-app) zijn waarop de data kan worden bekeken.
* Er moet een **rapport** zijn "documentatie.md" waarin je de verschillende onderdelen van je eindopdracht toelicht.
* Maak een "requirements.txt" bestand aan met alle nodige Python-pakketten om je project te kunnen draaien.

Je mag vrij kiezen welk toestel je gebruikt om data vast te leggen. Wel moet er minstens één **Raspberry Pi** in de flow aanwezig zijn.

## Scoring

Indien je enkel de geziene leerstof in de eindopdracht verwerkt, kan je een maximale score van **16/20** behalen. Om een hogere score te behalen, moet je andere technologieën, zoals een andere cloudprovider, in de eindopdracht verwerken.

## Hardware

* Raspberry Pi 4 Model B - 1GB RAM
* SD-kaart (128GB)
* Voeding voor Raspberry Pi

## Software

* Raspbian OS Lite (64-bit)
* Python 3.11 of hoger
* Streamlit
* Azure Cloud Services (IoT Hub, Functions, Storage Account, Machine Learning)
* Lokale AI-modellen (bijv. TensorFlow Lite, ONNX Runtime)
* Cloud AI-modellen (Azure Machine Learning)
